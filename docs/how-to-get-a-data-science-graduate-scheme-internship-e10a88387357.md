# 学习一些数据科学

在申请实习或研究生项目之前，你应该学习基本的数据科学和机器学习知识。

我不是说你需要成为强化学习方面的高手，或者稳定扩散的专家，但了解数据科学的基本内容会帮助你申请职位时脱颖而出。它还会帮助你确认，数据科学是你长远规划中的职业。
