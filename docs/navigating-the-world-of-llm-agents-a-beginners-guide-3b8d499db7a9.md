# 简介

这篇文章讲述的是如何让 LLM（大语言模型）独立解决复杂任务。

人类在不断吸收信息、做出决策、采取行动、观察变化，然后做出下一个决策的能力上非常出色。我们的一生就是一个永无止境的**观察、思考和行动的链条**。我们希望将这一概念应用于 LLM，使其能够持续做出新的决策，从而逐步接近复杂问题的解决方案。
