# #1\. 简单线性回归简介

预测分析的领域广阔，但在其核心是线性回归——最简单的方法，用来理解数据趋势。

虽然它扩展到多个变量时可能会让人觉得有些艰难，但今天我们的重点将专注于简单线性回归。
