# 什么是计算思维？

计算思维是一个解决问题的框架，它通过算法将任务分解为我喜欢称之为“原子任务”的小任务。它涉及设计一个逐步的算法方法来解决问题，识别相似性和低效之处，并评估每个步骤的相对重要性。
