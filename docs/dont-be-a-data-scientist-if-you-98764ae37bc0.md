# 如果你没有成长心态，就不要做数据科学家

在职业生涯的早期阶段，出现冒充者综合症是非常常见的——我们常常怀疑自己是否有资格获得这份工作。它在技术行业尤其普遍，因为这是一个快速发展的领域。一些去年使用的算法可能已经过时。每个人都在追逐最新的流行词汇，这会给求职者和在职人员带来很大的压力。

如果不保持成长心态，就无法摆脱这种不配感……
