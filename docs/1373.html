Background/Context

I was thrilled to learn that the memory feature of ChatGPT was released on April 29, 2024, as a subscription-only service. Excitedly, I tried it out immediately. I couldn’t stop thinking about how this memory feature was implemented, along with the important considerations and trade-offs involved. By following the principles and solution outline of GenAI ML system design interviews, I’ll share my thoughts and considerations as an interview candidate for designing the ChatGPT memory feature in this post.
