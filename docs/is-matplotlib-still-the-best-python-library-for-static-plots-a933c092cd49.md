# 介绍

在文章标题中，您可能注意到我特别提到了**静态图表**。

## 静态与动态

尽管仪表板和交互式图表在探索和展示数据方面极为重要，但静态图表始终是必不可少的。

报告、技术论文、文章以及所有印刷材料总是需要静态图表。因此，能够快速、轻松地制作清晰、逻辑性强且美观的静态图表仍然是绝对必要的。考虑到这一点，本文将主要关注静态图表。
