# 1. 碰撞图：跟踪排名随时间变化

碰撞图是一种特殊类型的折线图，用于可视化多个类别的排名随时间变化的情况。它的y轴表示类别排名，因此可以显示各类别如何随着时间的推移相互“碰撞”并上下浮动。因此，它非常适合展示类别之间的竞争。
