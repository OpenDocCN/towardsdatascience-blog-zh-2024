## 我应该掌握多少 SQL 知识？

SQL 是数据科学家的基础语言，因此你应该掌握它。SQL 的优点是它比 Python 更容易学习，因为它是一个非常小巧的语言。

我有一篇文章解释了成为一名优秀入门级和中级数据科学家所需掌握的 SQL 知识，强烈推荐你阅读。
