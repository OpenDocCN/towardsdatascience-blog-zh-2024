# **适合谁？**

鉴于如今对 AI 感兴趣的背景范围非常广泛，我尽力使这篇指南具有普遍的可访问性。然而，没有哪份指南可以适合所有人。以下是我考虑的几个特定群体。

+   **技术专业人士**尝试提升技能以促进职业发展

+   **商业领袖**那些多年没有编写代码的人，试图跟上不断变化的技术环境

+   **创业者**构建 AI 原生产品

+   **学生**们尝试提升他们的技术 AI 技能
