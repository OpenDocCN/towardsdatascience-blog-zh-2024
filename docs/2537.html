If you are an OpenAI user, you might be aware that very recently they added an enhancement to their system prompt on the OpenAI Playground website.

Now, you have the option to ask OpenAI to generate a new, detailed and highly focussed prompt that you can use in place of your initial, simpler prompt.

To test this out, head over to the playground using the link below and log in with your openAI credentials if you’re not already.

https://platform.openai.com/playground/

You should see a screen like this.

If you look at the two icons on the right-hand side of the System Instructions input box, you’ll see one that looks like a 4-pointed star. This is the prompt generator and is what will turn your simple prompt into a highly focused and detailed one.
